# City-Mart-BI-Project
This project analyzes retail business operations of City Mart using Power BI. The objective is to support management decision-making through interactive dashboards and data visualization.

## Objectives
- Analyze sales and operational performance
- Monitor department performance
- Understand the Order-to-Cash process
- Provide data-driven insights for business improvement

## Tools & Technologies
- Microsoft Power BI
- Microsoft Excel
- Data Modeling
- KPI Reporting

## Dataset
The dataset contains retail operational data including:
- Sales transactions
- Department information
- Order processing data
- Business workflow details
(This dataset was used for academic project purposes.)

## Dashboard Features
- Sales performance overview
- Department-wise performance analysis
- KPI indicators for monitoring business performance
- Interactive filters and slicers for dynamic reporting

## Key Insights
- Identified sales performance trends across departments
- Detected variations in operational efficiency
- Highlighted potential areas for process improvement

- Data cleaning and preparation
- Data modeling and relationship building
- Dashboard design and visualization
- Business process analysis
